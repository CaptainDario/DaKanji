Deep Links

DaKanji supports https links and a custom scheme. Https links have the base scheme https://dakanji.app/app/ and the custom scheme is dakanji://.

Https handling is achieved on iOS and MacOS using universal links, on Android using app links and on Windows using apps for websites. However, it is not supported on Linux. The custom scheme is supported on all platforms.

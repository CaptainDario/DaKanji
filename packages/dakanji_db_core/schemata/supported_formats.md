### Conversion

There are some tools that can be used to convert between different dictionary formats:

* [yomitan-import : Yomitan Import allows users of the Yomitan extension to import custom dictionary files.](https://github.com/yomidevs/yomitan-import)
* [pyglossary : a tool for converting dictionary files aka glossaries](https://github.com/ilius/pyglossary)

If you now other converters [please let me know!](https://github.com/CaptainDario/DaKanji/issues/new)
